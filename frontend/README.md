# Freelance Marketplace Frontend

## Setup

1. Install dependencies: `npm install`
2. Start the app: `npm start`

## Features

- Home page with jobs and services
- User registration and login
- Role-based dashboard
- Post jobs (clients)
- Submit proposals (freelancers)
- Messaging system
